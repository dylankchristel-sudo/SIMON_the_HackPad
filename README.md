# SIMON

SIMON is a keypad designed to be usable in several applications. It has 5 keys, a rotary encoder, an OLED display, and 4 WS2812B LEDs. It uses KMK firmware.

# Features:

128x32 OLED Display  
EC11 Rotary encoder  
4 WS2812B AR LEDs

# CAD Model:

Two parts: a case and a lid. (IDK why my PCB rendered that way). The holes will be covered with keycaps and the rotary encoder.
